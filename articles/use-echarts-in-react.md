# この記事は
グラフ描画ライブラリの一つである[`Echarts`](https://echarts.apache.org/en/index.html)を使ってみます。
[デモが豊富にある](https://echarts.apache.org/examples/en/index.html)ので、作成したいグラフのイメージに近いものがあるか一度見てみると良いかもしれません。

# やってみる
## インポート
```
npm install echarts
```
